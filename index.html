Paul Fiterau

I am a postdoctoral researcher at Uppsala University. Previously, I completed my PhD at Radboud University Nijmegen before working as a Software Developer in industry for a brief period. To find out more about me and my background, check my CV.
My work involves developing effective and (preferably) automated techniques for testing stateful systems, particularly network protocol implementations. These techniques have so far been based on model learning or symbolic execution.

Research Interests

My research interests lie in:

  • network protocol testing
  • model learning algorithms and applications
  • symbolic execution
  • application of formal methods
  • conformance and model-based testing

Software Tools

Below are some of the notable tools I made significant contributions to (and possibly still am).

  • DTLS-Fuzzer - protocol state fuzzer for DTLS
  • tcp-learner - protocol state fuzzer for TCP
  • Tomte - learning framework for Register Automata using a CEGAR approach
  • Z3GI - learning framework based on SMT solvers, for a variety of formalisms
  • RALib - learning framework for Register Automata
I also made contributions to:
  • TLS-Attacker - popular framework for testing (D)TLS stacks
